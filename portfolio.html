Maville - Urban Works Management System

University of Montreal | Java Project

Project Overview

A comprehensive urban management platform addressing Montreal's public works coordination challenges. The system connects residents, service providers, and city officials to improve transparency and efficiency in infrastructure projects.

Key Features

  • Multi-role access (Residents, Providers, City Officials)
  • Customizable notification system
  • Real-time work tracking with Montreal's public API integration
  • Road problem reporting with priority management
  • Project bidding and approval workflow
